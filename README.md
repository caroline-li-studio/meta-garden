# Meta Garden 🌸

A beautiful digital garden preserving the memory of extinct plants and animals.

## About

Meta Garden is an interactive web experience where multiple extinct species overlap beautifully on the landing page. Clicking any species takes you to a detailed dictionary-style page with information about that organism.

### Features

- ✨ Beautiful overlapping design with floating animations
- 🌿 Interactive landing page - click any species to explore
- 📖 Detailed information cards for each extinct species
- 📱 Fully responsive design works on mobile and desktop
- 🚀 Ready for Vercel deployment

## Species Included

- **Dodo** (Mauritius, 1662)
- **Passenger Pigeon** (North America, 1914)
- **Thylacine (Tasmanian Tiger)** (Australia, 1936)
- **Steller's Sea Cow** (Bering Sea, 1768)
- **Saint Helena Olive** (Saint Helena, 2003)
- **Quagga** (South Africa, 1883)
- **Great Auk** (North Atlantic, 1844)
- **Franklin Tree** (USA, extinct in wild early 1800s)

## Development

It's just HTML/CSS/JavaScript - no build step required!

```bash
# Serve locally with Python
python3 -m http.server 8000

# Open http://localhost:8000
```

## Deployment to Vercel

1. Push this repo to GitHub
2. Connect Vercel to the GitHub repo
3. Deploy - that's it! Vercel will automatically serve the static files.
